    <h1>Code-Laboration Project</h1>

    <h2>Description</h2>
    <p>
        Code-Laboration is an interactive tool designed for teachers to present code examples during class. It allows teachers to input lines of code that are displayed in a stacked format, enabling students to follow along easily.
    </p>

    <h2>Features</h2>
    <ul>
        <li>Input multiple lines of code.</li>
        <li>Stacked display of code lines, with a maximum of 15 lines visible.</li>
        <li>Option to go to a specific line of code.</li>
        <li>User-friendly interface with buttons for easy navigation.</li>
    </ul>

    <h2>Technologies Used</h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>

    <h2>Getting Started</h2>
    <p>To get started with the project, follow these steps:</p>
    <ol>
        <li>Clone the repository: <code>git clone <your-repository-url></code></li>
        <li>Open the <code>index.html</code> file in your web browser.</li>
        <li>Use the input field to add code lines and click "Add Code" to display them.</li>
    </ol>

    <h2>Usage</h2>
    <p>
        Enter the desired code in the input area and click the "Add Code" button. The new code line will be displayed at the top of the code window, and if the limit of 15 lines is reached, the oldest lines will be removed.
    </p>
